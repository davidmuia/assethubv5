Instructions

Follow these steps to bulk import assets:

  1. Download the CSV template. Download Template
  2. Fill in the asset data. Ensure all required* columns are completed.
  3. Save the file as a CSV.
  4. Upload the file using the form on this page.

Important: The import is "all or nothing". If a single row contains an error, the entire import will be cancelled to prevent partial data entry.
